## SnapScholar
This is an AI-powered educational video summarizer with visual snapshots.

This is an applied languages model group prject in AI and Deep Learning Models Development course at Google and Reichman tech school.


Setup

1. Create a virtual environment

python3 -m venv venv


2. Activate the environment

source venv/bin/activate


3. Create your environment file

touch .env


Add the following to your .env file:

GOOGLE_API_KEY="XXX"


4. Install dependencies

python3 -m pip install -r requirements.txt

Running the App
streamlit run app.py

Running Tests
python test_topic_visuals.py

Deactivate the Environment
deactivate
